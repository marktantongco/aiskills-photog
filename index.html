00 — SYNERGY MAP

How Skills Connect

Skills compound. Mastery of foundational prompting enables effective consistency control; consistency enables reliable hybrid workflows; reliable workflows enable scalable agent orchestration.

FOUNDATION Technical Prompt Engineering + Photographic Literacy
CONSISTENCY LAYER Strategic Negation + Identity Preservation
REFINEMENT LAYER Post-Processing + Hybrid Workflows
ORCHESTRATION AI Agent Design + Production Deploy
01 — PROMPT ENGINEERING

Technical Prompt Engineering

Constructing prompts as structured blueprints, not keyword lists. The input layer for all AI generation.

Scaffold Method

Structured prompt order: Subject + Action + Lighting + Lens + Style + Quality

A woman walking in rain + cinematic lighting + 85mm f/1.8 + photorealistic + 4K

Information Priority

Front-load critical elements. AI weights early tokens more heavily

"Oil painting of a castle..." vs "A castle, oil painting..."

Photographic Vocabulary

Use precise technical terms over vague buzzwords

85mm lens, f/2.8, shallow depth of field

Camera Movement

Specify complex cinematography in text prompts

pull from close-up to wide, dolly zoom

Agent Patterns

Design prompts for multi-step agent workflows

Zero-shot, few-shot, chain-of-thought

Long-Horizon Video

Maintain narrative coherence across sequences

Scene 1: Close-up, golden hour. Transition: Cross-dissolve...
02 — PHOTOGRAPHIC LITERACY

Photographic Literacy

Reconstructing real-world physics to achieve believable, professional-grade results.

Lighting Patterns

Apply classic studio lighting setups

Rembrandt: key light 45° high, fill 1/4 power

Lens Selection

Choose focal length for desired perspective

85-135mm portrait, 35-50mm standard

Aperture Control

Use f-stop to control depth of field

f/1.4-f/5.6 shallow; f/8-f/32 deep

Native Resolution

Select output resolution based on use case

4K native for pores; avoid upscaling

Vertical Format

Compose for 9:16 short-form platforms

Center subject, negative space above

Advanced Rendering

Physics-based cues for realistic materials

subsurface scattering, ambient occlusion
03 — STRATEGIC NEGATION

Strategic Negation & Skin

Telling the AI what NOT to include to overcome unnatural perfection.

Negative w/ Weights

Exclude unwanted elements using weighted syntax

(plastic skin:1.4), (cartoon:1.3), (extra fingers:1.4)

Skin Realism

Prompt natural textures while negating artifacts

Prompt: pores | Negate: airbrushed, waxy

Anatomical Correction

Proactively target common AI errors

(fused fingers, double iris:1.4)

Temporal Consistency

Prevent feature drift across video frames

--cref + error recycling

Drift Correction

Feed artifacts back for stability

Generate → identify → retrain → regenerate
04 — IDENTITY

Identity & Consistency

Maintaining specific characters, styles, and narratives across generations.

Seed Locking

Fix noise pattern for controlled variations

--seed 12345

Reference Tools

Platform-specific reference systems

--cref char.jpg --sref style.jpg --cw 80

Character Weight

Fine-tune reference influence scope

--cw 0 (face) to 100 (full)

Multi-Reference

Combine multiple references

char1 + char2 + background + prop

Visual-Script Alignment

Score frames vs storyboard

Flag >15% deviation

Agent-Based

Specialized agents for coherence

DirectorAgent coordinates tools
